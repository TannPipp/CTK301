About Me

Hi there! I'm Tanner Pipp and this is my Portfolio. For a little bit of background, I've always thought that I would be involved with technology in some way growing up, and getting to be a part of this program has shown me what opportunities there are. Making this for my web design class has been an interesting and fun experience! I got to learn about the beginnings of HTML and CSS coding, getting to see the potential of making websites for future projects and potential careers that could involve web design. In This I'll be showing the different kinds of projects that I've gotten work on while I've been at ISU, as well as looking at the things I've done before ISU that helped me realize the things I'd like to spend my time doing.
